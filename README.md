<h1 align="center"> GET A PET 🐶 </h1>

<p align="center" id="objetivo"> Projeto com intuito de facilitar adoção de animais de pequeno porte, Conectando o doador ao adotador sem precisar sair de casa!🏡 </p>

<p align="center">✅ Projeto finalizado ✅</p>

## 🛸 Features

- [x] Cadastro e Edição De Dados De Usuários.
- [x] Login Com Verificação.
- [x] Cadastro e Edição De Dados Dos Pets.
- [x] Dashboard Retornando Todos Os Animais Cadastrados Pelo Usuário.
- [x] Autenticação com JWT.

## ⚙️ Rodando o BackEnd (servidor)
```bash
# Clone esse repositório na sua maquina.
$ git clone https://github.com/jeysonvieira/get-a-pet.git

# Acesse a pasta do projeto pelo terminal/cmd.
$ cd get-a-pet

# Instale as dependências do projeto.
$ npm install

# Execute a aplicação.
$ npm start

# A aplicação iniciará na porta:5000 - Acesse <http://localhost:5000>
```
